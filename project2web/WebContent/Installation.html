In this tutorial we will be using Eclipse IDE

STEP 1: Maven Installation

To install maven follow the below steps:
1. Open Eclipse
2. Go to Help -> Eclipse Marketplace
3. Search by Maven
4. Click "Install" button at "Maven Integration for Eclipse" section
5. Follow the instruction step by step
Once the maven is installed goto windows preference
We should be able to list maven on the left list
