What is Spring?

Spring is an open-source framework that provides infrastructure support for developing Java enterprise applications. It provides support for different application architectures like transactional data and persistence, messaging and web. It allows developers to choose, as per their requirement, from a variety of sub-frameworks and modules supported by it, including Spring MVC.

Key features

POJO

Spring supports application development from "Plain Old Java Object", commonly referred to as POJO development structure.

Dependency Injection

Dependency injection provides the capability of separating the initialization of objects from the class which consumes those objects as dependencies. For example - Consider a simple class Store, whose constructor requires an object of another class Item to initialize its object- public class Store { private Item item; public Store(Item item) { this.item = item; } } This is an example of DI where we have separated initialization of Item object from initialization of Store object. So, Store class has a dependency over Item class and this dependency is injected into Store object by calling class. Spring core container modules provide support for dependency injection using XML configurations as well as using annotations like @Bean, @Configuration and @Autowired. For example - @Configuration public class AppConfig { @Bean public Item item1() { return new ItemImpl1(); } @Bean public Store store() { return new Store(item1()); } } Here @Configuration on AppConfig class indicates that it contains some beans which are item1 and store. This is an example of constructor-based DI, where when constructor of Store class is invoked, a dependency is injected utilizing another bean (item1 bean) since it is an argument for Store class constructor.
