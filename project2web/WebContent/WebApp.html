Overview of College Course Registration Web Application

In order to demonstrate how the Spring MVC framework and the Hibernate ORM software add-on can be used together our group developed a web application for enrolling in college courses. This web application allows users to login to their account or enroll and setup a student account. After logging in or registering users are directed to their profile page where they can edit their profile information and view the available courses that they can enroll in. Student records are saved in a persistent MySQL database that the application is connected to.
